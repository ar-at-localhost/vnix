* Basic Features
** Workspaces
   - Create / Rename / Delete a workspace on the fly.
   - Persist workspaces in specs files.
   - Navigation keys
      - ~<leader>K~ Previous workspace
      - ~<leader>H~ Next workspace
      - ~<leader>g~ First workspace
      - ~<leader>G~ Last workspace
   - Other keys
      - ~<leader>"~ Kill
   - Configurations
     - Name
     - Current working directory
     - DEV
       - Orgmode
***** Presets
     Presets are templates which you can use for spinning workspaces
     with predefine layouts.

     - ~blank~ A workspace with single tab / single pane.
     - ~DEV~ A workspace that has several tabs and each with one or more panes
       for effectively managing development workspaces (repositories.)
       - Editor: A tab that runs your favorite editor (mine/default ~nvim~).
       - Term: A single pane terminal for quick tasks.
       - V Split / H Split / Grid: 3 tabs with several panes.
*** Tabs
   - Create / Rename / Delete tabs on the fly.
   - Persist tabs alongside workspaces in specs files.
   - Navigation keys
      - ~<leader>L~ Tab right 
      - ~<leader>H~ Tab left
      - ~<leader>0~ Tab first
      - ~<leader>$~ Tab last
   - Sort / Swap tabs with left / right
      - ~<leader>)~ Swap with tab right 
      - ~<leader>(~ Swap with tab left
   - Other keys
      - ~<leader>'~ Kill
   - Configurations
     - Name
     - Current working directory
***** Presets
     Presets are templates which you can use for spinning tabs 
     with predefine layouts.

     - V Split: A 2 pane tab with vertical split between (|).
     - H Split: A 2 pane tab with horizontal split between (-).
     - Grid: A 4 pane tab like a grid (-|-).
**** Panes
   - Create / Rename / Delete panes on the fly.
   - Persist tabs alongside workspaces in specs files alongside workspaces / tabs.
   - Tree splits (right / left) 
   - Resize panes
   - Navigation keys
      - ~<leader>l~ Pane right 
      - ~<leader>h~ Pane left
      - ~<leader>k~ Pane upper
      - ~<leader>j~ Tab lower
   - Resize keys
      - ~<leader><~ Increase left
      - ~<leader>>~ Increase right
      - ~<leader>^~ Increase up
      - ~<leader>v~ Increase down
      - ~<leader>;~ Kill
   - Other keys
      - ~<leader>;~ Kill
   - Configurations
   - Configurations
     - Name
     - Current working directory

* Orgmode
  Vnix has first class support for orgmode (via [[https://github.com/nvim-orgmode/orgmode][Orgmode Nvim]]).
  It enables you manage your tasks and notes very effectively.
